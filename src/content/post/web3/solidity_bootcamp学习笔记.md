# 11.16 第一节课

今天主要讲了区块链的基本原理，与 web2 的区别。了解到在 web3 的开发中，比较注重的是合约（类似传统后端）和前端，而前端的技术栈主要是 react 那一套（这一点在后面收集到的案例中也有体现，大部分都是 nextjs 来实现的）

由于我从未接触过 web3，听下来还是挺吃力的，不过感觉挺有趣。打算在课程之余，另外找一些课程来学习和实践，做一些感兴趣的东西，以及给开源项目提 pr。
